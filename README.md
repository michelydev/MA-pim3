# 📚 Plataforma de Avaliações Educacionais

Projeto desenvolvido para o Projeto Integrado (PIM) do curso de Análise e Desenvolvimento de Sistemas.

## 📖 Sobre o Projeto

A Plataforma de Avaliações Educacionais foi idealizada para apoiar o processo de ensino e aprendizagem por meio da aplicação de avaliações digitais, permitindo que alunos realizem questionários e que professores acompanhem o desempenho obtido.

Além do desenvolvimento da solução, o projeto envolveu etapas de análise, modelagem, prototipação e planejamento, simulando um cenário real de desenvolvimento de software.

---

## ✨ Funcionalidades

### 👨‍🎓 Aluno

* Realizar avaliações
* Visualizar avaliações disponíveis
* Receber devolutivas e resultados das avaliações

### 👨‍🏫 Professor

* Criar avaliações
* Consultar avaliações aplicadas
* Analisar o desempenho dos alunos

### 👨‍💼 Administrador

* Criar usuários
* Editar usuários
* Listar usuários cadastrados
* Gerenciar acesso ao sistema
* Realizar autenticação (login)

---

## 🎨 Interface e Experiência do Usuário

A interface foi projetada com foco em:

* Facilidade de navegação
* Organização visual das informações
* Experiência intuitiva para os usuários
* Responsividade para diferentes dispositivos
* Prototipação das telas utilizando Figma

---

## 📐 Modelagem do Sistema

Durante o desenvolvimento foram elaborados os seguintes artefatos:

* Diagrama de Casos de Uso
* Diagrama de Classes
* Modelo Entidade-Relacionamento (MER)
* Levantamento de requisitos
* Documentação do sistema
